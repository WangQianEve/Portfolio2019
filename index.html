Index Sense: Use the keyboard surface as a touchpad.
Advisor: Prof. Anind Dey Collaborator: Julian Ramos
We explored a new space for interaction on laptops -- keyboard surface. We used computer vision technology to detect the 3D position of the finger tip, thus supporting using the keyboard surface as a track pad. Fitts law experiments shows the speed of our design is much faster than track pad, almost as fast as the mouse.

Detect jitters in mobile phone transition animations and model users' perception of it.
Advisors: Prof. Yuanchun Shi, Prof. Chun Yu Collaborator: Mingyuan Zhong Sponsor: Huawei
To guarantee a smoother mobile user experience, we built an animation evaluation system. It uses computer vision technology to detect jitters and uses machine learning technology to predict users' perception of the jitters, thus deciding how to improve the user experience. The machine learning model is built based on data from an online user study of more than 2000 people.

Advisors: Prof. Yuanchun Shi, Prof. Chun Yu Collaborator: Yudong Chen, Guangbin Chen
