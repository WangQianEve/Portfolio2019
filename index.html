Index Sense: Keyboard Surface as a Trackpad
Advisor: Prof. Anind Dey Collaborator: Julian Ramos
We explored a new space for interaction on laptops -- the keyboard surface. We employed computer vision technology to detect the 3D position of the fingertip, thus converting the keyboard surface to a trackpad. Fitts law experiments show that the speed of our design is much faster than the trackpad, almost as fast as the mouse.

Detection and User Perception Modeling of Jitters in Mobile Phone Transition Animations
Advisors: Prof. Yuanchun Shi, Prof. Chun Yu Collaborators: Mingyuan Zhong Sponsor: Huawei
To guarantee smoother user experience on mobile phones, we built an animation evaluation system. The system utilizes computer vision technology to detect jitters and employs machine learning methods to predict users’ perception of jitters, providing insights on how to improve the user experience. The machine learning model is trained using data from an online user study of more than 2000 people.

Gesture-Controlled Animation Generation
Advisors: Prof. Yuanchun Shi, Prof. Chun Yu Collaborators: Yudong Chen, Guangbin Chen
We built an animation generation tool to experiment with how well can one use their hands to control the movement of the avatar. The system is built on Unity, with Leap Motion as the gesture recognition component. Results show that using gesture alone is not accurate enough and is tiring for the user.
